# Reflection

## What went well?
This week I successfully converted my terminal based Python application into a Streamlit web app. I was able to create a working user interface with input fields and dropdowns, and deploy it to Streamlit Cloud. The app runs in the browser and allows users to build a PC and see the results.

## What could you have done better?
I could have planned the transition from terminal input to a web interface more clearly before starting. I also could have debugged more efficiently by checking logs earlier instead of guessing what was wrong.

## What could I have done better?
The AI could have provided a more structured step-by-step approach earlier, especially explaining how Streamlit differs from traditional Python input/output. More explanation about deployment issues would have helped reduce confusion.

## What have we learned?
I learned how to convert a Python program into a web application using Streamlit, how to deploy apps to the cloud, and how to debug common issues like blank screens and incorrect file setups. I also learned the importance of building features step-by-step instead of all at once.
